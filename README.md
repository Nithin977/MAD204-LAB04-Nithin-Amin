\## Project Description

This Android application allows users to create, view, update, and delete notes using a local Room database.  

It also includes:

\- A background Reminder Service that sends notifications.

\- A BroadcastReceiver that listens for connectivity changes.

\- Full CRUD operations with RecyclerView.



Technologies used:

\- Kotlin

\- Room Database

\- RecyclerView

\- Services

\- BroadcastReceiver

\- Notifications



\## Features

\- Add, edit, delete notes

\- Local database persistence using Room

\- Snackbar undo for delete

\- Background reminder notification

\- Network connectivity detection



\## How to Run

1\. Clone the repository

2\. Open in Android Studio

3\. Sync Gradle

4\. Run on emulator or physical device
